# Recommended Deep-Learning Sequence

| Phase | Focus                                                                |
| ----- | -------------------------------------------------------------------- |
| 1     | Learn digital communication basics: clock, baud rate, frames, duplex |
| 2     | Study UART and USART first (simplest concepts)                       |
| 3     | Learn SPI and I2C comparatively                                      |
| 4     | Learn CAN for reliability and networking concepts                    |
| 5     | Learn USB after understanding protocol stacks                        |
| 6     | Compare all protocols using practical embedded projects              |
| 7     | Analyze logic-analyzer captures and timing diagrams                  |
| 8     | Build projects using each protocol on microcontrollers               |

# Most Important Conceptual Differences

| Protocol | Clock        | Duplex       | Topology       | Speed      | Reliability | Complexity |
| -------- | ------------ | ------------ | -------------- | ---------- | ----------- | ---------- |
| SPI      | Synchronous  | Full         | Master-slave   | Very High  | Medium      | Medium     |
| I2C      | Synchronous  | Half         | Multi-drop bus | Medium     | Medium      | Low        |
| CAN      | Synchronous  | Multi-master | Network bus    | Medium     | Very High   | High       |
| UART     | Asynchronous | Full         | Point-to-point | Low-Medium | Low         | Very Low   |
| USART    | Both         | Full         | Point-to-point | Medium     | Medium      | Medium     |
| USB      | Synchronous  | Host-device  | Tiered star    | Very High  | High        | Very High  |
